핀스톡(PinStock) 개인정보처리방침

최종 업데이트: 2026년 4월 7일

1. 개요

핀스톡(이하 "앱")는 주식·금융 뉴스 및 시장 동향을 제공하는 서비스입니다. 본 방침은 앱이 수집하는 정보, 사용 목적, 보관 방법을 설명합니다.

2. 수집하는 정보

가. 앱이 직접 수집하는 정보

  • 관심 키워드, 지역, 알림 레벨, 다크모드 등 사용자가 직접 입력한 설정값
  • 북마크한 뉴스 기사 목록
  • 읽은 알림 내역
이 데이터는 오직 기기 내부(로컬 저장소)에만 저장되며, 외부 서버로 전송되지 않습니다.

나. 제3자 SDK가 수집하는 정보

  • Google AdMob: 맞춤 광고 제공을 위해 광고 식별자(GAID/IDFA), 기기 정보, 앱 사용 패턴을 수집할 수 있습니다.
    → Google 개인정보처리방침

3. 인터넷 권한 및 외부 API 사용

앱은 다음 외부 서비스에 네트워크 요청을 보냅니다.

  • 네이버 검색 API — 뉴스 기사 수집
  • NewsAPI — 글로벌 금융 뉴스
  • Yahoo Finance — 실시간 시장 지수
  • KRX Open API — 국내 상장 종목 목록
  • Google Gemini API — AI 시장 요약

이 과정에서 기기의 IP 주소가 각 서비스 서버에 전달될 수 있습니다. 앱은 사용자를 식별하기 위한 어떠한 정보도 해당 서버에 전송하지 않습니다.

4. 권한 사용 목적

  • 인터넷(INTERNET): 뉴스 및 시장 데이터 수신
  • 네트워크 상태(ACCESS_NETWORK_STATE): 오프라인 여부 감지
  • 알림 전송(POST_NOTIFICATIONS, Android 13+): 속보 및 시장 급변 알림 발송
  • 부팅 완료 수신(RECEIVE_BOOT_COMPLETED): 기기 재시작 후 백그라운드 뉴스 체크 작업 재등록
  • 배터리 최적화 제외(REQUEST_IGNORE_BATTERY_OPTIMIZATIONS): 15분 주기 백그라운드 알림의 안정적 동작 (사용자 동의 시에만 적용)

5. 광고

앱은 Google AdMob을 통해 배너 광고, 전면 광고, 앱 오픈 광고를 표시합니다. iOS 14 이상에서는 앱 추적 투명성(ATT) 동의 요청이 표시될 수 있으며, 동의를 거부해도 앱 사용에는 지장이 없습니다(비맞춤 광고로 표시됩니다).

6. 데이터 보관 및 삭제

앱이 로컬에 저장한 모든 데이터(설정, 북마크, 알림 내역)는 앱을 삭제하면 완전히 제거됩니다. 별도의 계정이나 서버 저장소를 사용하지 않으므로 추가 삭제 절차가 없습니다.

7. 아동 개인정보 보호

이 앱은 만 14세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 개인정보처리방침 변경

방침이 변경될 경우 이 페이지의 "최종 업데이트" 날짜를 갱신합니다. 중요한 변경 사항은 앱 내 알림으로 안내할 수 있습니다.
